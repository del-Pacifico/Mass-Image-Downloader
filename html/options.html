📘 How to Use & Optimize

Quick tips to help you get the best out of Mass Image Downloader.

What it does:
☕ This extension allows you to extract and download images in bulk, either from open tabs or grouped galleries. It supports automatic filtering, customizable filenames, batch control, and clipboard shortcuts.

🧰 Feature overview:
📸 Download images directly in tabs: Download open images in tabs filtering and downloading valid images (currently open in the browser window).
🌄 Extract images from galleries (with direct links): Extract images from galleries with direct image links (via HTML anchor tags).
🖼️ Extract visual galleries: Extracts images from galleries (without link), detecting by grouping and similarity of paths.
🔗 Extract Web-Linked Galleries: Analyzes images from linked galleries (to web pages), opening associated web pages, selecting the high resolution image found on each page (adding icon for download).
📋 Clipboard hotkeys: Press Ctrl+Shift+P or Ctrl+Shift+S to paste and set new prefix/suffix quickly from clipboard.

🚀 Performance tips:
🔴 Low-end PCs: Set Max images per batch = 1 or 2, avoid auto-closing tabs.
🟡 Mid-range PCs: Use batching (2–3), smart grouping, and keep visual filters enabled.
🟢 High-end PCs: Enable fallback grouping, open tabs concurrently (max 5–10), and allow all formats.

🔧 Troubleshooting:
• Ensure gallery images are large enough (check min width/height).
• Confirm image formats (JPG/PNG/WEBP) are enabled in File system section.
• Logs can be enabled under Debugging (choose log level).
• Reload the extension if options are not applied correctly.
