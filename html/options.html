üìò How to Use & Optimize

Quick tips to help you get the best out of Mass Image Downloader.

What it does:
‚òï This extension allows you to extract and download images in bulk, either from open tabs or grouped galleries. It supports automatic filtering, customizable filenames, batch control, and clipboard shortcuts.

üß∞ Feature overview:
üì∏ Download images directly in tabs: Download open images in tabs filtering and downloading valid images (currently open in the browser window).
üåÑ Extract images from galleries (with direct links): Extract images from galleries with direct image links (via HTML anchor tags).
üñºÔ∏è Extract visual galleries: Extracts images from galleries (without link), detecting by grouping and similarity of paths.
üîó Extract Web-Linked Galleries: Analyzes images from linked galleries (to web pages), opening associated web pages, selecting the high resolution image found on each page (adding icon for download).
üìã Clipboard hotkeys: Press Ctrl+Shift+P or Ctrl+Shift+S to paste and set new prefix/suffix quickly from clipboard.

üöÄ Performance tips:
üî¥ Low-end PCs: Set Max images per batch = 1 or 2, avoid auto-closing tabs.
üü° Mid-range PCs: Use batching (2‚Äì3), smart grouping, and keep visual filters enabled.
üü¢ High-end PCs: Enable fallback grouping, open tabs concurrently (max 5‚Äì10), and allow all formats.

üîß Troubleshooting:
‚Ä¢ Ensure gallery images are large enough (check min width/height).
‚Ä¢ Confirm image formats (JPG/PNG/WEBP) are enabled in File system section.
‚Ä¢ Logs can be enabled under Debugging (choose log level).
